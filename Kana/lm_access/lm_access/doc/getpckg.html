How to get the lm_access package from CVS

do the usual DØ software staff (for the details, look at Heidi's tutorials on DØ software)
    >      (only on d0mino)
    > setup D0RunII <version>  (lm_access package is not connected to any release, it is up to you to decide which release version to use)
    > setup d0cvs
    > newrel -t <version> <dirname>
    > cd <dirname>
    > addpkg -h lm_access
at the end you should find the directory lm_access in your release directory <dirname>
check Luminosity ID web page for the current valid version of lm_access package

Compilation of the lm_access package

create/modify your code;
in the release directory <dirname> type
    > gmake lm_access
if everything went fine you should find the lm_access library under the directory p1107/lib/< architecture> /

If you get some compilation error concerning lm_access_interface, add also this package to your working directory (addpkg -h lm_access_interface)

To run it, you will also need to setup d0_config
