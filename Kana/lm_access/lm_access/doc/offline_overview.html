Current Luminosity Access

Luminosity access is currently done via the package lm_access - C++ code which enables users to determine a luminosity for a given data sample/trigger.

Requires complete list of LBN's in the analysis sample, returns luminosity for good LBNS and list of bad LBNS.

Users must check to see if there is valid luminosity information for each event and reject events without it.
