Creating Parentage Information for data which is not part of a common sample.

Please check first that your data are in fact not part of a common sample, those often have pregenerated parentage files. Generating parentage files is VERY time consuming and you could save yourself a day by checking there first.

 

If your files are not part of a common skim you need to build your own parentage files

In the lm_access/scripts directory, there are three scripts makeFileParentageList.py, makeDimensionParentageList.py and makeDatasetParentageList.py, make certain these are in your PATH and PYTHONPATH.

type 

makeDatasetParentageList.py <your dataset definition>

For each file in the sam dataset dataset it will provide a file <filename>.parentage which contains the lbns for that file.

These files take some time to generate but can be reused or stored in a common area for your group. They will not change as once a file is in sam,the metadata will not change.
